# 实验2 逻辑回归/支持向量机（分类对比）

- 做了啥：主要是在乳腺癌和鸢尾花数据集上，对比逻辑回归和 SVM 的分类效果。重点考察了 SVM 的不同核函数（线性、多项式、RBF）的表现，还花了不少时间调 C 和 gamma 这两个关键参数，看看它们怎么**影响决策边界。**
- 遇到的问题：数据里有的类别样本多有的少，导致模型容易偏向多数类；超参数 C 和 gamma 稍微变一点效果差很多，网格搜索跑起来特别慢；而且核 SVM 在特征多或者样本多的时候，计算量大得让人头秃。
- 解决方案：针对类别不平衡，用了 class_weight 或者重采样来平衡权重；调参时先大范围粗搜锁定区域，再细搜；特征必须标准化，不然距离计算会歪；实在跑不动的时候，就先采样子集或者用线性 SVM 近似一下。
- 启发：核函数决定了决策边界长啥样，线性核简单直观，但非线性核（像 RBF）能弯弯绕绕拟合得更好；正则强度 C 是个平衡木，控制间隔宽窄和容错率；评估的时候别光看准确率，ROC 和 PR 曲线更能说明问题。
